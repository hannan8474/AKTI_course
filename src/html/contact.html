How to create github

  1. Check hidden items from view
  2. Open github or create account
  3. Goto github repositories
  4. Open git bash terminal in VS code
  5. First run git init command in terminal
  6. Then git add . command
  7. After this we run git commit -m command
  8. Now run git push command
  9. Your git repository is created
  10. git push -u origin main
